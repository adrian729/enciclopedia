# MIX ’N’ MATCH TRAIL MIX

> MAKES | About 2 cups · TOTAL TIME | 5 minutes · TOTAL COST | $1–$10 (depending on your choices)

## Ingredients

| Quantity | Ingredient |
|---|---|
| **¾ cup, any nut or seed or a combo** | **NUTS & SEEDS** |
| | Peanuts |
| | Almonds |
| | Walnuts |
| | Cashews |
| | Hazelnuts |
| | Pecans |
| | Sunflower seeds |
| | Pepitas (pumpkin seeds) |
| | Other nuts or seeds |
| **½ cup, choose 1 or a combo** | **CHEWY BITES** |
| | Raisins |
| | Dried cranberries |
| | Dried figs, chopped |
| | Dried blueberries |
| | Dates, chopped |
| | Prunes, chopped |
| | Other dried fruit |
| **½ cup, choose 1 or a combo** | **CRISPY BITES** |
| | Mini pretzels |
| | Popcorn ([Popcorn](cooking/books/fast-easy-cheap-vegan/snacks/diy-microwave-popcorn.md)) |
| | Cereal |
| | Potato chips |
| | Banana chips |
| | Broken or small crackers |
| | Sesame sticks |
| **¼ cup, choose 1, optional** | **EXTRA PUNCH** |
| | Vegan chocolate chips |
| | Crystalized ginger |
| | Wasabi peas |
| | Shredded coconut |
| | Broken or small cookies |
| | Seaweed rice crackers |

## Preparation

1. To a medium bowl or sealable container, add ¾ cup nuts and/or seeds, ½ cup chewy bites, ½ cup crispy bites, and an extra punch ingredient, if desired. Mix together and enjoy.

**QUICK TIPS:** 1. I love buying roasted and salted nuts for an extra flavor punch. 2. If your grocery store carries it, freeze-dried fruit is another fun add-in!

**MAKE AHEAD:** Store in an airtight container in your pantry for up to 1 month.
