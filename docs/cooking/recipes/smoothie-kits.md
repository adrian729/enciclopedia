# Smoothie Kits

> Total: 5 mins · Yield: 1 serving · Cost: $1–$5 · Equipment: blender

## Ingredients

| Quantity | Ingredient |
|---|---|
| | **Fresh or Frozen Fruit (1 Cup, Choose 1 or a Combo)** |
| | Blueberries |
| | Strawberries |
| | Mixed berries |
| | Cherries |
| | Peaches |
| | Pineapple |
| | Kiwi |
| | Apple |
| | Banana |
| | **Add Some Creaminess (Choose 1)** |
| 1 | fresh or frozen banana |
| ½ cup | fresh or frozen mangoes |
| ½ cup | chopped fresh or frozen avocados |
| ½ cup | silken tofu |
| ¼ cup | whole rolled or quick oats (gluten-free if preferred) |
| | **Amp It Up (Choose 1–3, Optional)** |
| 1 handful | fresh or frozen spinach |
| 1 handful | fresh or frozen kale leaves or stems |
| 1 tablespoon | peanut butter or other nut butter |
| 1 tablespoon | cocoa powder |
| ½ cup | coffee ice cubes |
| 2 teaspoons | instant espresso powder |
| 1 small handful | fresh mint leaves |
| ¼ teaspoon | cinnamon |
| ¼ teaspoon | [Pumpkin Pie Spice](cooking/recipes/pumpkin-pie-spice.md) (or store-bought) |
| 2–4 | dates (for sweetness) |
| ¼ cup | raisins (for sweetness) |
| | **Liquid (1–2 Cups, Choose 1)** |
| | Plant-based milk, such as soy or oat |
| | Water |
| | Fruit juice |
| | Cooled coffee |
| | Cooled herbal tea |
| | Coconut water |

## Preparation

1. For each smoothie kit, add 1 cup fruit of choice (or combination of fruits), one creamy ingredient, and one to three amp-it-up ingredients, if desired, to a small bag or container, and freeze (reusable freezer bags work well). Make several kits at a time so one is always ready to go.
2. When ready to blend, dump one of the frozen smoothie kits into a blender and add 1 cup liquid of choice, adding more liquid as needed to reach the desired consistency.
